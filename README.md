## Setup
Clone this repo to your desktop and run `npm install` to install all the dependencies.

## Install

```sh
npm install
```

## Usage

Add ".env" file and update the values/settings to your own

Production
```sh
npm start
```
Development Build
```sh
npm run start:dev
```

## Run tests

```sh
npm run test
```
